Fashion POS es un sistema integral de punto de venta desarrollado para tiendas de ropa, permitiendo gestionar inventario, registrar ventas y generar reportes financieros detallados. Administra productos con variantes como talla, color y categoría, manteniendo control automático de stock en tiempo real.

Cada venta genera un comprobante digital y actualiza inventario mediante transacciones seguras en base de datos, garantizando consistencia e integridad. Además, el sistema permite análisis de métricas comerciales diarias y mensuales.

El proyecto aplica principios de normalización de bases de datos, transacciones ACID, arquitectura basada en API REST y separación clara entre frontend y backend. Está diseñado para ser escalable y adaptable a distintos entornos comerciales.
