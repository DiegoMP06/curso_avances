Enterprise Tracking CMS es una plataforma web diseñada para centralizar la gestión y el seguimiento estratégico de empresas. El sistema permite registrar, actualizar y monitorear información corporativa clave como estados operativos, métricas financieras, historial de actividad y relaciones comerciales dentro de una arquitectura moderna y escalable.

La aplicación incorpora un dashboard analítico con indicadores visuales y reportes exportables, facilitando la toma de decisiones basada en datos. Implementa control de acceso basado en roles (RBAC), asegurando que cada usuario tenga permisos definidos según su perfil organizacional.

A nivel técnico, el proyecto utiliza una arquitectura desacoplada basada en API REST, base de datos relacional normalizada en PostgreSQL, autenticación mediante JWT y validaciones tanto en frontend como backend. Se aplican buenas prácticas de seguridad, separación de responsabilidades y diseño modular, garantizando mantenibilidad y escalabilidad.
